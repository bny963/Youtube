# PR Title
動画アップロードのバリデーション強化とエラー表示の実装

## 概要
動画投稿時のファイルサイズや形式の制限がないため、巨大なファイルによるサーバー負荷や、非対応形式によるサムネイル生成エラーが発生するリスクがあります。

## ゴール
- バックエンドでファイルサイズ（50MB以内）と拡張子（mp4, mov等）を制限する。
- フロントエンドでバリデーションエラーの内容を表示する。
- アップロード中のローディング状態を可視化し、二重送信を防止する。

## 実装内容
- バックエンド（Laravel）の`VideoController@store`にバリデーションルール追加。
- エラーメッセージの日本語化・カスタマイズ。
- エラーレスポンスをJSON形式で返す。
- フロントエンド（Next.js）ではファイル選択時の事前チェック、エラーメッセージの表示コンポーネント実装、アップロード中のローディング状態表示、送信ボタンの無効化を実装。
